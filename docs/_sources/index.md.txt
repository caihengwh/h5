实战 H5 && Javascript
==========================


**培训后，你将学会**
   
   1. 掌握H5、CSS、JavaScript的基本技能；
   2. 学会GitHub的操作，并且学会在GigHub上搭建自己的网站；
   3. 学会使用BootStrap设计网页；
   4. 学会使用JavaScript，完成CRUD的操作；
   5. 搭建个人RestFul API 服务，学习如何与HTTP交互；
   6. 学习如何测试及部署自己的应用程序。


**培训班的宗旨**

   1. 为公司储备全栈开发人才，提升项目交付能力；
   2. 培训及普及全栈知识，协助学员提高编码技能。没有证书，只交流code技术。
   
**课程的设计及思想**

   1. 质量：引进国外培训机构全栈认证培训教学经验；
   2. 效率：每节课有预定的学习目标, 每个知识点有个综合的sba练习；
   3. 有效性：授课知识来自项目的实践内容，现场编码授课，同时迭代更新教程；
   4. 完整性：教学知识递进式设计，上节课输出的内容是下节课的输入；
   5. 学习方法：主动学习，非灌输式的。
