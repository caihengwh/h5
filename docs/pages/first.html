教学计划¶

第一课¶

本篇内容是为初学者准备的知识，内容包括：

课前预习：

  1. Github的使用，参考link： https://www.bootcss.com/p/git-guide/

  2. 预习本课件第1章节

课堂讲解：

  1. 使用Github部署自己的 WebApp；

  2. Web基础3要素：HTML + CSS + JavaScript（第2课讲）；

  3. H5的入门和使用；

  4. CSS的flex入门和使用；

  5. 开发自己的个人页面；

课后练习：

  1. 准备自己的开发环境；

  2. 完成练习：https://github.ibm.com/fsd-training/h5-course/tree/master/assignment/lession1

  3. 部署自己的练习作品到Github上；

  4. 更新学员练习进度表：https://pages.github.ibm.com/fsd-training/h5-course/
